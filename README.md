# 🛠️ Tooly - All-in-One Private Web Tools

Tooly is a lightweight, high-performance suite of web utilities designed with a **privacy-first** approach. Every tool runs 100% locally in your browser—no data ever touches a server.

## 🚀 Live Demo

Check it out at: [https://tooly.vercel.app](https://tooly.vercel.app)

## 💎 Features

- **Privacy Focused**: All processing happens client-side.
- **SEO Optimized**: Fully indexed for search engines and AI discovery.
- **Fast & Minimalist**: Built with pure HTML, CSS, and Vanilla JS.
- **No Dependencies**: Lightweight and robust.

## 🧰 Included Tools

- 🕒 World Time Converter (with smart search)
- 📊 BMI Calculator
- 📅 Date & Weeks Counter
- 🔗 QR Code Generator
- 💰 Real-time Currency Converter
- 🏆 Giveaway Winner Picker
- 📄 PDF Text Extractor
- 💡 Brand Name Generator

## 🛠️ Tech Stack

- HTML5 / CSS3 (Modern Flexbox & Grid)
- Vanilla JavaScript (ES6+)
- Schema.org (JSON-LD) for AI Search Optimization
